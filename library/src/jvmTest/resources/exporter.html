How to use the exported palette

✅ GIMP (desktop)

  1. Click Download .gpl for GIMP above
  2. Save the file as my-hsl-palette.gpl
  3. Open GIMP → Windows → Dockable Dialogs → Palettes
  4. Right-click in the Palettes window → Import Palette…
  5. Select your .gpl file → Import
  6. Your palette appears instantly and is ready to use with the Color Picker or Bucket tool

🌐 Online photo editors (Photopea, Pixlr, etc.)

  1. Photopea (recommended): File → Open → select the .gpl file (or drag the PNG)
  2. Pixlr / Canva / Figma: Use the PNG swatch – most editors let you load an image as a custom palette or eyedropper from it
  3. Or simply paste the copied hex list into any palette importer that accepts text

The palette is exported exactly as shown (254 colors + 2 transparent padding cells in the visual grid). The .gpl and PNG contain only your real colors in the exact order you provided.
